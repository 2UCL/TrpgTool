【はじめに / なにこれ】
 このアプリケーションは 「TRPG を円滑に進められたら良いなぁ」っていう
適当な目的で作られたWindows専用のツールです。()
なお製作者がクトゥルフをメインでしかやったことないので、
ほぼ(というか完全に)仕様がクトゥルフ専用になっています。()

【旧バージョンからのアップデート】
 旧 trpgt_v***.exe を新しいexeファイルに置き換えてください。

【パッケージ内容】
 Package_TT12a ┬ ARCIVE.zip		- これまでのバージョンをまとめてあります。
               │               	 気になった時や、必要な時にどうぞ。
               ├ hspinet.dll		- インターネット接続やJSON解析等に必要なファイルです。
               │               	 削除するとアプリケーションが正常に動作しません。
               ├ trpgt_v120a.exe	- アプリケーションです。こちらを起動してください。
               └ readme.txt		- 今あなたが見ているファイルです。説明書です。

【実行方法】
 解凍(展開)したパッケージ内の オレンジアイコン、「trpgt.exe」 を起動するのみ。
ちなみに tucd ファイルをドラッグ&ドロップして「exeで開く」をしても開きます。
([プログラムから開く]等でもOK)

【操作方法】
 初めて起動した際は 空データ というタイトルで、
既に初期値のキャラシートがテンプレートで表示されます。

◎キャラシート編集、作成
　　「File」 → 「キャラシート編集」をクリックして、
　　ウィザード画面の指示に従ってデータを入力するとキャラシートを生成して適用してくれます。
　　ファイルにも保存しておけるので後から呼び出すことも可能です。

◎マニュアルダイス、技能値判定
　　下部に「ManualDice」ウィンドウがあり、そこから 1d1～6d999 の範囲で指定して
　　「ROLE!」をクリックすればダイスを振る事が可能です。
　　なお、「1d3,1d4,1d6,1d100」 をクリックしても同等のダイスを振ることも可能です。
　　また、「RAND!」をクリックすると内部の乱数生成機構を再構成します。
　　結果が気に食わない時などにどうぞ。()

　　画面中央の「技能一覧」から、
　　コンボボックスから 技能分類 を、リストボックスから 技能 を選択して「Role」ボタンを押すか、
　　クイックダイス一覧から選択すると、選択した技能の値判定を 1d100 で行います。
　　結果によって下部の「ManualDice」ウィンドウの光り方が変わります。
　　クリティカル(OK+)、ファンブル(Fail+) の範囲設定は「設定」から行えます。

◎ダイス履歴
　　上記の方法でロールしたダイスの履歴はすべて右側の「Dice History」ウィンドウで記録しています。
　　ダイス履歴は保持してないので起動するたびに真っ新になります。
　　「Clear」ボタンを押すと履歴を消去することができます。
　　「Sure?」と聞かれるので「Yes」を押してください。

◎クイック値編集(HP,MP,SAN)、SANC
　　ステータスの HP,MP,SAN ボタンを押すとキャラシート編集画面までいかなくても
　　値を編集することができます。
　　SAN は「ROLE!」を押すことで SANC(SAN値チェック) をすることも可能です。

◎キャラクター詳細
　　「Detail」ボタンを押すとSTRやCONなどの基本能力値や、
　　キャラクタープロファイルを閲覧することが可能です。

◎ファイルメニュー
　　メイン画面から File をクリックすると表示されます。
　　キャラシートの読込、保存やアプリケーションの設定、
　　外部ファイルのインポート(vampire-blood.netからのJSON取得) や、
　　エクスポート(txt, csv ファイル)ができます。

【セーブファイルについて】
　アプリケーションで使用する保存ファイルは以下の通りです。
	setting.json	- アプリケーションの設定情報を保存しています。
	temp.tucd	- リジューム機能が有効な時に生成されるファイルです。
			 そのままTUCDファイルとしても利用可能です。
	*.tucd		- ユーザが任意で使用できるキャラシートファイルです。

　　※ v0.8 の TUCDファイルは互換モードとしてそのまま使用可能です。
　　　通常通り、 ファイルから開く を実行して読み込ませてください。

【エラーコード / 困ったときは】
 　アプリケーションにエラーコードが出ている場合
	ChkSumError	- 読み込んだTUCDファイルに異常があります。
			 ファイルの修復が必要な場合は報告をお願いします。
	FormatError	- 読み込んだファイルがTUCDファイルではありません。
	JS-1		- 指定したキャラシートが存在しないなど、使用できないファイルです。
	JS-2		- クトゥルフ用のキャラシートを指定してください。
	JS-3		- キャラシートに設定間違いがあります。
	NW-**		- ネットワークに接続する段階で問題が発生しました。
	NR-**		- ファイルの受信中に問題が発生しました。
	SB-**		- プログラム起動中に問題が発生しました。
	SY-**		- プログラムの内部で問題が発生しました。
			 ウィンドウタイトル欄の E,W,L,I の数値と一緒に報告をお願いします。

　　※ ** 部は任意の数値が入ります。

 　何かダイアログが出てきた 
　　内部エラーが発生しました	- 上記 SYエラー と同様です。
　　Runtime Error! Program: …	- ダイスを振った際に出てきた場合は、残念ながら、
  　				 現状 解決が難しいエラーです...  もう少し間隔を置いて
				 ダイスを振るようにしてくださると幸いです。

　 困ったときは / お問い合わせ
